# Hi, I'm Vansh Arora. Welcome to my digital playground.

This isn't just a portfolio; it's a showcase of my passion for creating beautiful, interactive, and meaningful web experiences. I built this to demonstrate not just what I can do, but *how* I think about technology and design.

---

## My Vision: Code as a Craft

I believe that code is more than just instructions for a computer. It's a craft. This portfolio is my canvas, where I blend cutting-edge technology with a user-centric design philosophy. My goal is to create applications that are not only functional but also delightful to use.

---

## What's Inside?

This portfolio is packed with features that showcase my skills and creativity:

*   **✨ Interactive UI:** From magnetic buttons to a skill orbit, every element is designed to be engaging.
*   **🚀 Stunning Visuals:** A particle background and 3D tilt cards create a modern and immersive experience.
*   **✍️ Typing Terminal:** A nod to my love for the command line, this terminal types out my story.
*   **🎨 Creative Animations:** Subtle animations and effects that bring the portfolio to life.

## See it for Yourself

Want to take a look under the hood? Here's how you can run this project locally:

1.  **Clone the repo:** `git clone https://github.com/GiggaCat/About_Me`
2.  **Install dependencies:** `npm install`
3.  **Add your API keys** to a `.env` file.
4.  **Run it:** `npm run dev`
